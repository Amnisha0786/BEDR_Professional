# BEDR Professional

## Overview

BEDR Professional is a UK-based healthcare platform designed for eye care management and collaboration between multiple user roles, including doctors, optometrists, clinics, readers, patients, and administrators. The platform provides interconnected dashboards with real-time updates, secure data handling, medical scan management, and payment integration.

I worked primarily on the frontend development of the application, building scalable and reusable UI architecture while integrating complex backend services and healthcare-related workflows.

---

## User Panels

The application includes multiple interconnected panels:

* Doctor Panel
* Optometrist Panel
* Clinic Panel
* Reader Panel
* Patient Panel
* Admin Panel
* Optimized iPad View for clinical workflows

---

## Key Features

* Role-based authentication and authorization
* Real-time updates across all panels using WebSockets
* Eye scan and medical file handling
* DICOM scan viewing integration
* Stripe payment integration
* Notifications system
* Pagination and optimized large data handling
* Responsive and reusable UI components
* Secure API integration with RSA encryption
* Cross-panel live synchronization of patient data
* File upload, download, and management functionality

## Major Challenge & Learning

One of the biggest technical challenges in this project was integrating the MedDream DICOM Viewer for patient eye scans. Since medical imaging workflows require accurate rendering, secure handling, and smooth performance, integrating the viewer into the frontend while maintaining responsive UI and real-time synchronization significantly improved my problem-solving and frontend integration skills.

This project strengthened my expertise in:

* Production-level frontend architecture
* Real-time application development
* Healthcare application workflows
* Complex API integrations
* State management and optimization
* Secure data handling
* Performance optimization for large-scale applications
